[UAVREL Dataset v1.0]

[Dataset for UAV Video Dynamic Relationship Understanding]

Overview

[UAVREL是针对无人机视频动态目标关系理解的大规模数据集，包含50段高清无人机视频，标注了目标间的时空关联]

  • Number of samples: [50 videos]
  • Annotation types: [relationship]
  • License: CC-BY 4.0

Download

Example Code (Python)

# Load annotation file
import json
